# aneskurtovic.com

Personal portfolio website for Anes Kurtovic — Software Engineer.

Built with [Astro](https://astro.build).

## Tech Stack

- **Framework:** Astro 5
- **Styling:** Vanilla CSS with custom properties
- **Fonts:** Inter, JetBrains Mono
- **Deployment:** Static site

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project Structure

```
src/
├── components/    # Astro components (Nav, Hero, Experience, etc.)
├── data/          # TypeScript data files (experience, projects, skills, etc.)
├── layouts/       # Page layout with global styles
├── pages/         # Route pages
└── styles/        # Global CSS and animations
```
